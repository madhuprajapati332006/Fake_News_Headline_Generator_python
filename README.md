# 📰 Fake News Headline Generator

## 📖 Overview
The Fake News Headline Generator is a simple Python project that creates random fictional news headlines by combining different subjects, actions, and places. It is designed as a beginner-friendly project to practice Python concepts such as lists, functions, and the random module.

---

## 🎯 Objectives
- Learn Python fundamentals.
- Practice using the random module.
- Understand list manipulation.
- Build a fun text-generation project.

---

## ✨ Features
- Generates random fake news headlines.
- Uses predefined lists of subjects, actions, and places.
- Produces a different headline each time the program runs.
- Easy to modify by adding your own data.

---

## 🛠️ Technologies Used
- Python 3
- Random Module
